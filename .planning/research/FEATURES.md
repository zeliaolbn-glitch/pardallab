# Benchmarking de Recursos - IdeaFlow (2025/2026)

## Análise Competitiva
O mercado de 2025 foca na transição fluida de **Vendas/Ideia** para **Execução/Projeto**.

### Recursos Essenciais (Must-haves)
1.  **Arquitetura de Dados Unificada**: O registro de uma ideia deve evoluir diretamente para um projeto sem perda de contexto.
2.  **Automação de Workflow**: Mudança de status (ex: "Ideia" -> "Iniciada") deve disparar notificações ou criar tarefas iniciais automaticamente.
3.  **Integração Omnichannel**: Centralização de contatos (WhatsApp/Telegram) vinculados a projetos específicos.
4.  **Dashboards Visuais**: Uso de Kanbans e Cards com estados claros para evitar o "limbo das ideias".

### Tendências de Integração de IA
- **Assistentes Generativos**: IA que popula listas de tarefas baseadas na descrição do projeto.
- **Agentic AI**: Agentes que podem sugerir ferramentas baseadas no contexto (Lovable AI integration).
- **Análise de Risco**: IA identificando gargalos potenciais (ex: "Muitos projetos em andamento, pouca validação").

### Diferenciadores do IdeaFlow
- **Foco em "Ideias Pequenas"**: Captura rápida com posts curtos, similar a redes sociais, para baixar o custo de entrada da ideia.
- **Status Granular (Funil de Lançamento)**: De "Ideia" até "APK", cobrindo todo o ciclo de vida do desenvolvimento indie/startup.
- **Manual de "Ferramentas Sugeridas"**: Curadoria integrada para o usuário não travar na escolha da stack.

## Sugestões de Melhoria no Plano Original
- Adicionar uma **Timeline de Evolução** no card do projeto.
- Implementar **Tags de Categoria** para filtrar ideias por nicho (SaaS, E-commerce, Mobile, etc.).
